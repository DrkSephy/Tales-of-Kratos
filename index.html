Controls:
    Left Mouse Button : Open Menus, Start Game
    X : Jump
    ENTER : Open Status Screen
    ESC : Cancel menus
    Down + A : Cast Heal
    Down + S : Cast Lightning
    Down + F : Cast Fireball
    Down + D : Cast Wind
    A : Demon Fang
    Right/Left Arrow + A : Lightning Blade
    Right/Left Arrow + S : Sonic Thrust
    Up Arrow + S : Hell Pyre
